# Expense-Tracker-Python

A simple Expense Tracker built using Python

## Features

- Add Expense
- View Expenses
- Show Total Expense
- Delete Expense
- Search Expense by Category
- Save Data in File
- Load Data Automatically

## Concept Used

- Function
- Lists
- Dictionaries
- File Handling
- Loops
- Conditional Statements
